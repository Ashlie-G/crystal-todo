### Todo App - Spider Gazelle

#### Todo's

**View All Todo's**

Method: GET
Path: /todos

**Create Todo**

Method: POST
Path: /todos

**View Todo**

Method: GET
Path: /todos/:id

 **Update Todo**

 Method: PATCH
Path: /todos/:id

 **Delete Todo**

 Method: DELETE
Path: /todos/:id

### Dependencies

Prior to running locally, please run:
```shards install```

To run development environment run:
``` crystal main.cr```

### Testing

- Run all tests:
```crystal spec```
